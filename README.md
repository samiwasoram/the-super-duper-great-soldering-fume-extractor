# the-super-duper-great-soldering-fume-extractor
ANTI CANCER SOLDERING FUME DIFUSER/FAN/THINGIE
ANTI CANCER SOLDERING FUME DIFUSER/FAN/THINGIE!!!
Esentially your run of the mill soldering fume difuser (like any that'd you find on Amazon). It will use a PC fan with an off and off switch, as well as carbon foam sponge. The case was designed in Onshape. It doesn't have a mounting mechanism, it uses a back panel to lean onto. Meant to be lightweight and space-conscious when I'm soldering, also 5v USB so I can easily plug it in on my desk. I plan to find a random USB wire in my house for this as well.
